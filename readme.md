# Config自定義加解密

## 簡介

使用aspnet_regiis.exe對web.config區塊進行加密
加密後若要解密必須在同一個IIS且提供同一金鑰

## 功能說明

1. web.config加解密
2. 建立RSA加密，並提供匯出入加密金鑰

## 選項說明

* (a). 一般加密
* (b). 一般解密
* (1). 建立金鑰容器
* (2). 移除金鑰容器
* (3). 匯出金鑰容器
* (4). 匯入金鑰容器
* (5). 金鑰容器加密
* (6). 金鑰容器解密
* (7). 建立IIS權限
* (8). 移除IIS權限
* (9). WEB.config
